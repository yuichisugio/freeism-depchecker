1. GitHub API空、GitHub CLIを使用して、指定したライブラリのGitHub Export SBOMを取得する
gh api -H "Accept: application/vnd.github+json" -H "X-GitHub-Api-Version: 2022-11-28" "/repos/facebook/react/dependency-graph/sbom"

2. 取得したバージョン、パッケージ名、登録名を指定して、依存ライブラリごとに、ホスティングサービスのowner/repoを取得
curl 'https://api.deps.dev/v3/systems/NPM/packages/scheduler/versions/0.23.0'
